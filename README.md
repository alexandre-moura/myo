# Myo

**Myo** is an advanced hypertrophy training companion app designed for serious lifters and gym-goers. It helps you randomize and track advanced training techniques like myo-reps, cluster sets, rest-pause, and more â€” all from a sleek, mobile-first interface.

---

## Features

- Randomizes advanced training techniques for experimentation and discovery each session
- Saves technique history for easy reference
- Smooth, fast UI powered by Expo, NativeWind, and Moti

## Tech Stack

- **React Native (Expo)**
- **Zustand** for global state

- **NativeWind** for Tailwind-style styling
- **Moti** for animations
- **TypeScript**
